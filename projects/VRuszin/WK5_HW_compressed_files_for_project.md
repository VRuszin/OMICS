## How I changed my project .fastq files into .fastq.gz files and added to github:

```bash
seqkit fq2fa filename.fastq -o solution/filename.fasta.gz
git add
git commit -m ""
git push
pull request
```


What my file size was before: 34.8 GB
What my file size was after: 3.9 GB
